แอปชื่อ Horoscope เป็นแอปดูดวงที่สามารถสลับกองไพ่ได้ทุกครั้งที่กดปุ่มstartเข้าไปหน้าดูดวง เเละเวลาดูดวงจะดูได้เเค่
ครั้งละ1ใบ(ใส่algorithmไว้)เเละถ้าจะดูเพิ่มให้กดตรงรูปถังขยะเพื่อล้างข้อมูลเเล้วจะสามารถกดเลือกไพ่ได้ใหม่ได้ เเละถ้ายังไม่ได้กดกลับ
ไปหน้าเเรกเเล้วกดไพ่ใบเดิมซ้ำก็จะได้ไพ่ใบเดิม เเละตรงมุมบนขวาของแอปจะมีไอคอนอยู่2อัน อันเเรกนับจากทางซ้ายจะเป็น
ข้อมูลเพิ่มเติมว่าการดูดวงมาจากไหน เเละมีปุ่มcredit เพิ่มดูเครดิตของรูปเเละอื่นๆ ส่วนไอคอนที่2จะเป็นประวัติการกดไพ่ที่เคยกดได้ตอน
ที่ดูดวงในครั้งที่เข้าเเอป ถ้าออกเเล้วเข้าใหม่ข้อมูลจะหายเพราะไม่ได้ต่อAPI ข้อมูลไพ่ทั้งหมดจะอยู่ในไฟล์carddata ซึ่งมีทั้งหมด
